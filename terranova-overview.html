Sécurité
Mesures de protection en place
🛡
Edge middleware (proxy.ts)
Protection /admin/* et /compte/* au niveau Vercel Edge avant tout rendu. Vérification session Supabase + rôle admin + IP whitelist.
📋
CSP + Headers HTTP
CSP stricte, HSTS, X-Frame-Options, X-Content-Type-Options, Referrer-Policy, Permissions-Policy, COOP, CORP.
⏱
Rate limiting email
Max 5 emails/heure/IP sur les 4 routes de contact. Table Supabase email_rate_limits avec RLS. Retourne 429 si dépassé.
🧹
Validation & sanitisation
Longueurs max, regex email RFC, trim, escHtml sur toutes les routes email. XSS corrigé dans MapCanvas (données Overpass API).
🗺
Anonymisation IP (RGPD)
IPs tronquées avant stockage dans map_ad_events : IPv4 → dernier octet à 0, IPv6 → 48 bits conservés.
🔑
Server Actions sécurisées
Toutes les actions vérifient getUser(). Actions admin appellent checkAdmin(). Impersonation protégée contre l'escalade de privilèges.
🎣
Stripe webhook signé
Vérification constructEvent() + STRIPE_WEBHOOK_SECRET vérifié explicitement (plus de non-null assertion).
🤖
Anti-bot & déduplication pubs
Détection UA bot, rate limiting DB-based 30 req/min, déduplication 1 événement/(ad_id, IP anonymisée, type, jour).
🔍
SEO & Qualité
Infrastructure de référencement et expérience utilisateur
🗺
Sitemap dynamique
Toutes les annonces publiées + profils chercheurs actifs + pages statiques. Auto-mis à jour.
🤖
robots.txt
Indexation autorisée sur les pages publiques. /admin, /compte et /api sont exclus.
🖼
OG Images dynamiques
Image de partage générée (1200×630) avec photo du bien, titre, prix et branding Terranova.
🚫
Page 404 personnalisée
Page branded avec liens de retour vers la carte et les annonces.

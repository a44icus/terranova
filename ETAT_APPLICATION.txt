Stack technique
───────────────
  • Next.js 16 (App Router, Server Components, Server Actions)
  • Supabase (base de données PostgreSQL + Auth + Storage)
  • MapLibre GL (carte interactive open-source)
  • Tailwind CSS v4
  • Stripe (abonnements Pro)
  • Resend (emails transactionnels)
  • Zustand (état global côté client)
  • TypeScript

  2. CARTE INTERACTIVE  (/carte)
════════════════════════════════════════════════════════════════

  Carte
  -----
  • MapLibre GL avec plusieurs styles : rue, satellite, sombre, clair
  • Marqueurs personnalisés par type (vente/location) avec prix affiché
  • Clustering automatique des marqueurs (supercluster)
  • Outil lasso : sélection polygonale à main levée sur la carte
  • Géolocalisation de l'utilisateur (bouton dédié)
  • Rayon de proximité configurable autour d'un point

  Filtres
  -------
  • Type : vente / location
  • Catégorie : appartement, maison, bureau, terrain, parking, local
  • Prix max
  • Surface min/max
  • Nombre de pièces
  • Options : parking, terrasse, cave, piscine, jardin, ascenseur…
  • Classe DPE (A à G)
  • Ville / département
  • Synchronisation des filtres dans l'URL (partage de recherche)

  Sidebar biens
  -------------
  • Liste et vue grille commutables
  • Tri : prix croissant/décroissant, surface, date
  • Cartes biens avec photo, prix, surface, ville, type
  • Mise en favori depuis la liste

  Popup détail bien
  -----------------
  • Photo principale + galerie
  • Prix, surface, pièces, DPE, options
  • Score de quartier calculé à partir des POI environnants
  • Points d'intérêt proches : restaurants, écoles, santé, transport…
  • Bouton itinéraire (calcul de route depuis la position utilisateur)
  • Bouton favoris
  • Bouton comparer (jusqu'à 3 biens simultanément)
  • Lien vers la fiche complète

  Panneaux flottants
  ------------------
  • Favoris : liste des biens sauvegardés
  • Comparateur : tableau comparatif côte à côte (max 3 biens)
  • Biens à proximité : rayon paramétrable autour d'un point

  8. PUBLICATION  (/publier)
════════════════════════════════════════════════════════════════

  • Vérification préalable côté serveur : limite d'annonces du plan
    (si limite atteinte → page de blocage avec CTA "Passer Pro")
  • Formulaire 3 étapes :
    1. Informations : type (vente/location), catégorie, titre,
       description, prix, surface, pièces, SDB, étage, DPE,
       options, meublé, neuf/VEFA, référence agence (si pro)
    2. Localisation : adresse, ville, code postal + géocodage
       automatique (Nominatim/OpenStreetMap), option position
       approximative (cercle plutôt qu'épingle exacte)
    3. Photos : upload multiple (JPG/PNG/WebP), limite selon le plan,
       réordonnancement, désignation de la photo principale
  • Soumission en brouillon ou directement en attente de modération
  • Page de confirmation après publication


════════════════════════════════════════════════════════════════
  9. PAIEMENT / ABONNEMENTS  (Stripe)
════════════════════════════════════════════════════════════════

  Plans disponibles
  -----------------
  • Gratuit     : 3 annonces, 5 photos/bien, visible 30 jours
  • Pro Mensuel : 50 annonces, 20 photos/bien, visible 30 jours
  • Pro Annuel  : illimité, 20 photos/bien, visible 365 jours

  Flux Stripe
  -----------
  • Checkout Session (mode subscription) avec locale française
  • Codes promo acceptés
  • Webhook sécurisé :
    - checkout.session.completed → activation du plan
    - invoice.payment_succeeded  → renouvellement
    - invoice.payment_failed     → log (Stripe réessaie auto)
    - customer.subscription.deleted → retour au plan gratuit
    - customer.subscription.updated → mise à jour expiration
  • Portail client Stripe (gestion, annulation, factures)
  • Expiration automatique : plan dégradé à "gratuit" si plan_expire_at
    est dépassé (vérifié à chaque visite de /compte/*)

Formulaire de contact agent sur /vendeur/[id] Nouveau composant ContactAgentForm en accordéon. Envoie un email à l'agent via Resend avec reply-to automatique. Route API /api/email/contact-agent. Affiche un état de succès après envoi.
Feat
Bouton de partage sur /vendeur/[id] Nouveau composant ShareButton utilisant l'API Web Share native avec fallback clipboard. Feedback "✓ Lien copié" après partage.

Seed SQL données de test agences Script supabase/seeds/agences_test.sql : 4 réseaux (Century 21, Orpi, Laforêt, GuideHome sans agent), 6 agents affiliés dans 6 villes différentes, 2 indépendants. DO block avec UUIDs fixés pour idempotence. Mot de passe : Test1234!.

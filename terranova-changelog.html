29 avril 2026
Corrections plan illimité & build Vercel
Bug critique plan admin, dropdown clippé et erreur de build Vercel au déploiement.
Fix
Plan "illimité" repassait en Gratuit immédiatement Bug dans isPlanExpired() : !expire_at → return true traitait null comme "expiré" au lieu de "illimité". Le layout /compte dégradait alors le plan en gratuit et l'écrivait en base. Corrigé : null sur plan pro = illimité (pas expiré).
Fix
Dropdown PlanGrantButton clippé par overflow-hidden Panel débridage clippé par le conteneur de la liste. Passage de position: absolute à position: fixed avec coordonnées calculées via getBoundingClientRect() au moment de l'ouverture.
Fix
Build Vercel — crash si RESEND_API_KEY absente new Resend(key) au niveau module levait une exception au chargement si la variable d'environnement n'était pas définie, bloquant la collecte des pages. Déplacé dans le handler POST (instanciation lazy).

Fix
Agents affiliés invisibles sur la page réseau Bug PostgREST : .eq('reseau_id', uuid) retournait 0 résultats. Workaround : fetch global + filtre JS. Colonne ville absente du SELECT causait un retour null global.
Fix
Comptes de test impossibles à utiliser 3 problèmes cumulés : hash bcrypt avec mauvais facteur de coût, auth.identities manquantes pour le provider email, seed ignoré par ON CONFLICT DO NOTHING à cause du trigger handle_new_user.
